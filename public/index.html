What You Get

Multi-LLM Support

Claude, GPT-4, Gemini, Ollama. Switch providers without code changes. Your API keys, your cost control.

Knowledge Graph

Neo4j-backed entity storage. Agents accumulate knowledge across missions. Automatic relationship extraction. Vector search built in.

Mission Orchestration

DAG-based workflows with checkpointing and crash recovery. Chain agents, tools, and conditions. Resume where you left off.

Full Observability

OpenTelemetry tracing, Langfuse for LLM debugging, Prometheus metrics. See exactly what your agents are thinking.

K8s-Native Scaling

Health probes, HPA scaling, graceful shutdown. Agents and tools run as pods. Scale horizontally out of the box.

Type-Safe Tools

Protocol Buffer I/O for all tool interactions. Automatic schema validation. Build tools in any language.
