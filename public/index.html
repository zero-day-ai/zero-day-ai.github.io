How It Works

Agents That Reason

Not scripts with if statements. Agents run an Observe-Think-Act loop, query knowledge graphs, and decide what to do next. Claude, GPT, Gemini, or local models via Ollama.

Tools Via Redis Queues

Agents dispatch work to tool pods. kubectl, curl, terraform, nuclei, whatever you need. Type-safe Protocol Buffer I/O. Horizontal scaling via HPA.

GraphRAG Knowledge

Neo4j stores everything agents learn: hosts, services, vulns, credentials, attack paths. Knowledge persists across missions. Agents get smarter over time.

Why This Matters

SaaS security tools can't touch your internal network. They can't see what's behind your firewall. They definitely can't run in your air-gapped environment. Gibson deploys where your infrastructure actually lives.

  • ✓ Runs on any K8s: EKS, GKE, AKS, k3s, OpenShift, air-gapped clusters
  • ✓ Your data stays in your network. Period.
  • ✓ Build agents with the Go SDK. Ship them as containers.
  • ✓ Full observability: OpenTelemetry, Langfuse, Prometheus

What People Build

Security Testing

Continuous pentesting agents that find vulns, chain exploits, and write findings with full provenance. Runs in CI/CD or on a schedule.

LLM Red-Teaming

Prompt injection, jailbreaks, data extraction. Built-in payload management with encoding transforms. Test your AI before someone else does.

Infrastructure Ops

Agents that troubleshoot K8s issues, analyze logs, respond to incidents. Delegate to sub-agents. Chain complex workflows.
